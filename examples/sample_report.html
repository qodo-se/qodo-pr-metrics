Qodo always performs a thorough review. This pace is your team’s counterfactual — how thoroughly you’d have reviewed the same code in‑house.
Outlier trim
Some PRs are huge but require little review — vendored libraries, multimedia & static assets, formatter sweeps, lockfile & snapshot updates. Trim them so the headline stays defensible.
